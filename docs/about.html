For this assignment, I have created a website dedicated to the basics of how the Internet works. This includes the following:

  • Domain Names and ICANN
  • IP Addresses, Packets, and Routing
  • DNS
  • HTTP and HTTPS Protocols
  • W3C, HTML, and CSS

The major goals of this assignment was to create a website (with HTML, CSS, Bootstrap, and Git) as well as learn about the basics of how the Internet works. As I progress further in IS 601 - Web Systems Development, I become more acquainted and familiar with creating web pages in HTML and CSS with Bootstrap and making commits and pushing these changes to GitHub. Learning about how the Internet works is also vital to know since I will be working primarily with computers in my future career. Many people don't know how the Internet works–it almost seems like magic–but there is a very real science that went into its creation. The pages featured on this website will seek to simplify the science of these mechanics into something that's a little more understandable, yet accurate.
