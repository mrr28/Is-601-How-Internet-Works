W3C

W3C stands for the World Wide Web Consortium. Founded by Tim Berners-Lee and located in Cambridge, Massachusetts, W3C's purpose and mission is to maintain and develop the standards for the World Wide Web. Some of the standards that W3C has developed include the use of HTML (hyper text markup language) and CSS (cascading style sheet) for the layout of webpages on the Internet. Maintaining the same standards for web pages, and the World Wide Web in general, will ensure that on all web browsers that the websites will be accessed and will function the same. The current CEO of W3C is Jeffrey M. Jaffe.

HTML and CSS

HTML and CSS are the two computer languages that create web pages on the Internet. HTML is meant to contain code for the content of the web page, including any text, images or videos, tables or graphs, etc. All of the most important content that's meant to be read goes in here. Some formatting can be made to text and other elements here as well. CSS on the other hand is meant to style the content on a web page. It doesn't have to be used whenever creating a web page but can be very helpful for editing the appearance of the web page by being able to easily change the colors, grid layout, font size and font family, and so on. As HTML is the standard language for accessing web pages, it can be useful for website developers to become familiar with the languages.

Below is a side by side comparison of what some HTML code looks like and how it looks when that code is being run. There are many ways to edit the content of a web page and format it to whatever a coder's heart desires, but these are some of the basic and common ones.

The key to using HTML is the use of tags. Tags are enclosed by '<' and '>' and within the tag is something that formats the content in one way or another. An example of this is h1, which is the largest setting for a header for text. There are even up to six header tags, starting from h1 to h6, that can be used to format headings. Another example tag is p, which sets text as a paragraph or body text.

The way CSS is used is it's normally created a separate document in the code and calls upon the tags used in HTML to be formatted. An example of this is if different header tags are being used (such as h2 or h4), then they can be automatically formatted using CSS. This eliminates the need to individually edit the tags in the HTML file itself. Tags are called upon in the CSS and one will set whatever appearance or layout settings they would like. This includes font color; size; and family, background color, alignment of some text or an image, set margins or borders, etc.
